# DARK_MYSTERY_STORY_BUILDER
A fun python mini-project that generates mystery stories using random templates and customizable word choices.

## This project combines:

- Random story generation.
- User interaction.
- File handling.
- Creative story telling.
  
## Features

- Random story generation.
- Manual custom story building.
- Saves stories to a text file.
- Random suggestions for each input.
  
## Technologies Used

- Python 3
- Colorama
- Random module
- File handling
- Functions
- Lists
- Loops and Conditionals
  
## Installation

 Install the require package:
 pip install colorama
 
## Run The Project

 python dark_story.py
